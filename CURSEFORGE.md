<p><strong>RS Polymorph</strong> is a NeoForge Minecraft mod that adds <a href="https://www.curseforge.com/minecraft/mc-mods/polymorph" rel="nofollow">Polymorph</a> compatibility for <a href="https://www.curseforge.com/minecraft/mc-mods/refined-storage-2" rel="nofollow">Refined Storage 2</a>.<br>It ensures seamless recipe selection within RS2 grids when multiple crafts are available.</p>
<p style="font-size: 1.2em;">✨ <strong>Created by <a href="https://github.com/vyrriox" rel="nofollow">vyrriox</a></strong> ✨</p>
<h3>✨ Key Features</h3>
<p>🛠️ <strong>Crafting Grid Support</strong> &mdash; Adds the Polymorph button to the standard RS2 Crafting Grid.<br>📜 <strong>Pattern Grid Support</strong> &mdash; Full compatibility for the Pattern Grid, perfect for autocrafting.<br>🎯 <strong>Precise Selection</strong> &mdash; Restores functional selection widgets to pick the exact recipe you need.<br>🔄 <strong>Dynamic Updates</strong> &mdash; Buttons appear and vanish dynamically based on recipe conflicts.<br>🧩 <strong>Seamless Integration</strong> &mdash; Designed to feel like a native part of Refined Storage 2.<br>🌐 <strong>Multi-lingual</strong> &mdash; Supports English and French (more languages coming soon).</p>
